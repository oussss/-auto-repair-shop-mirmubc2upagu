# The Golden Legend Auto Repair Website

A high-performance, single-page creative website for a premium auto repair business in Sharjah.

## Features
- **Modern Dark/Gold Theme:** Designed to reflect luxury, reliability, and industrial strength.
- **Custom Cursor:** Interactive mouse tracking script for desktop users.
- **Smooth Scroll Animations:** IntersectionObserver API used to reveal content on scroll.
- **3D Tilt Cards:** CSS 3D transforms manipulated by JavaScript on mouse movement for service cards.
- **Fully Responsive:** Mobile-first design architecture.
- **Bilingual Ready:** Fonts included for English (Outfit) and Arabic (Cairo).

## How to Run
1. Save `index.html`, `style.css`, and `script.js` in the same directory.
2. Open `index.html` in any modern web browser.
3. Ensure you have an internet connection to load Font Awesome icons and Google Fonts.

## Customization
- **Images:** The design uses CSS-based abstract graphics and Font Awesome icons to ensure zero-dependency loading. For production, replace `.hero-visual` and `.about-image` contents with real high-quality photography.
- **Colors:** Modify CSS variables in `:root` within `style.css` to adjust the branding palette.
